CSS - это язык оформления HTML-документов. За более подробной информацией сюда Так как он является вспомогательным языком, то существует несколько способов его добавления на web-страницу:

  • Внутренний <link rel="stylesheet" href="style.css">
  • Внутренний глобальный <style></style>
  • Внутренний локальный <span style="color: red"></style>

Синтаксис языва выглядит следующим образом: сначала указывается изменяемый обьект с помощью селектора После этого в фигурных скобках перечисляются через точку с запятой изменяемые свойства эдемента(цвет, размер, расположение и т.д.) , затем напротив каждого свойства указывается значение(red, left, 1px и т.д.). Например:

                    table {
                      width: 80%;
                      border-color: white;
                      border-style: groove;
                      border-width: 5px;
                    }
В ситуации когда изменить необходимо только неоторые элементы, следует пользаваться указателями класса: когда одни и теже теги принадлежат разным классам и оформляются по разному.
<table class="header"></table>
<table class="main"></table>
